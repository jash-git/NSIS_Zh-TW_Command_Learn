1.3 特性列表

  • 生成包含可執行安裝程式的外殼
  • 支援 ZLIB, BZIP2 和 LZMA 資料壓縮(檔案可個別壓縮或同時)
  • 支援卸載(安裝程式可以生成卸載程式)
  • 可自定義的用戶介面(對話、字型、背景、文字、檢測標記、圖檔等等)
  • 典型和新式嚮導介面
  • 完整的多語言，支援一個安裝程式有多個語言。自帶超過 35 種語言翻譯，但是你也可以建立你自己的
  • 頁面系統: 你可以新增標準嚮導頁面或自定義頁面
  • 用戶可以選擇安裝組件，樹形組件列表
  • 多個安裝配置(通常 最小、典型、完全)和自定義配置
  • 安裝程式自我驗證使用 CRC32 校驗和
  • 除了壓縮的安裝資料大小只需要很小的程式頭(預設為 34 KB)
  • 可以顯示文字或 RTF 格式的許可協議
  • 可以通過註冊表檢測目標目錄
  • 易用的插件系統(大多數插件用於建立自定義對話、網際網路連接、HTTP 下載、檔案打補丁、調用 Win32 API 等等)已經包含
  • 安裝程式可以達到 2 GB 最高限制
  • 可選性的靜默安裝模式用於自動安裝
  • 預處理器支援定義符號、宏、條件編輯、標準預設定義
  • 有意思的類似於 PHP 和彙編的編程體驗(包含用戶變量、堆棧、流程控制等等)
  • 安裝程式有它們自己的 VM，使你書寫的代碼可以支援:

    • 檔案解壓(可配置的覆蓋參數)
    • 檔案、目錄複製、重命名、刪除、搜尋
    • 調用插件 DLL
    • DLL/ActiveX 控件註冊、解註冊
    • 可執行外部程式(外部執行並帶有等待選項)
    • 建立快捷方式
    • 註冊表讀取、設定、搜尋、刪除
    • INI 檔案讀取、寫入
    • 慣用文字檔案讀取、寫入
    • 強大的字串和整數處理
    • 基於類名或標題判斷視窗
    • 用戶介面操作(字型、文字設定)
    • 發送 Window 消息
    • 通過消息框或自定義頁面與用戶交流
    • 分支、比較等等
    • 錯誤檢測
    • 支援重啟，包括重啟後刪除或重命名檔案
    • 安裝程式行為命令(比如顯示、隱藏、等待等等)
    • 在腳本可以使用用戶函數
    • 對用戶的動作回調函數

  • 對任何用戶完全免費。請看許可協議。
